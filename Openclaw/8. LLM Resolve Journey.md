### Critical Lesson
Always test API connectivity directly from VPS before configuring OpenClaw:
```bash
curl https://api.moonshot.ai/v1/models \
  -H "Authorization: Bearer YOUR_API_KEY"
```
If it returns a model list → endpoint works. If 401 → wrong key or wrong domain.
